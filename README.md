# Driving License Eligibility Checker

A simple Python program that asks the user about nationality and age,
then checks if the user is eligible to apply for a driving license.

## Live Demo
Try the program online on Replit:  
https://replit.com/@nairafouda68/mshrwaay-lsds-2#main.py

## Project Idea
- Ask the user for their nationality
- Ask the user for their age
- Check eligibility based on the given conditions
- Display the result to the user

## Eligibility Conditions
- Nationality must be Egyptian
- Age must be 18 or above

(Conditions can be modified as needed)

## How to Run
1. Run the Python file
2. Enter nationality
3. Enter age
4. View the result

## Technologies Used
- Python 3
